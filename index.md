### 1- Etude des aspects de sécurité d'Android - Audit d'une application Android, (2012): [Lien PDF](https://drive.google.com/file/d/0B2-Blrb7nOJocDVXaXlUQ1RxZDA/view?usp=sharing)

Le sujet de notre rapport c'est l'étude des aspects de sécurité du système d'exploitation Android ainsi savoir la manière dont ce système d'exploitation se présente, puis l'audit d'une application de ce dernier afin de savoir si le développeur de l'application a passé par les principes de développement sécurisé d'une application Android.

###### NB: Ce n'est pas le version final, c'est la raison pour laquelle vous allez trouver plein d'erreur d'orthgraphe ;)
